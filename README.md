## Assumptions and Trade Offs

Do to timeboxing the project styling is verylight and not covered everywhere. There is also a babel issue that is not resolved

- Kept it simple by using Sqlight
- Users can login and view books and leave a review.
- If the book has been reviewed error handling will pick up
- Users can leave reviews on existing books
- Users can edit and delete there reviews
- View templates could be simplified more.
